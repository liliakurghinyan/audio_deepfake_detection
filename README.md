This web application is the interface layer of the thesis project “Audio Deepfake Detection System”. The system accepts an uploaded audio file, preprocesses it, transforms it into a Mel Spectrogram representation, and sends it to a trained CNN model for binary classification.

Backend framework: FastAPI
Deep learning library: PyTorch
Feature extraction: Mel Spectrogram
Classifier: CNN
Supported input formats: WAV, MP3, FLAC, M4A, OGG, AAC
Automatic format conversion to WAV before prediction
This system is intended as an experimental and educational prototype for thesis research in AI-generated threat detection.
